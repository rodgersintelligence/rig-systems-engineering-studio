## Doctrine for A3 (the three hard rules above this line)

1. **Mandatory `policy.requiresApproval()`** — no auto-approval at this mode or above. AionUI with 72-hour timeout.
2. **Source-per-claim REQUIRED** — Python wrapper enforces. Every claim in the solution has ≥1 source URI.
3. **Mechanism map REQUIRED** — *"A strategy without mechanism is a wish."* Generated by Sonnet, validated against `MechanismMap` schema.

Plus:

4. **Hard budget caps**: `max_iterations` AND `max_cost_usd` per agent run. Cap is enforced by the wrapper, not the agent.
5. **Tool-call audit** — every MCP / tool invocation writes an audit row before and after.
6. **Red-team adversarial attacks** at A3.5 — Sonnet hits the solution with 5–20 attack modes.
7. **Diamond awareness**: D1 explores broadly (+5σ minimum on cognitive engines); D2 converges on mechanism; D3 measures drift against prior Brier baseline.

**Model**: Sonnet (Opus at L6).
**Cost**: $0.05 – $0.30.

```python
from ._budget import BudgetedAgent

def extract_intent(input: Multimodal, diamond: Diamond, max_iter: int = 5) -> Intent:

```

**Doctrine**: D1 explicitly does NOT commit to one intent. D2 commits + binds to mechanism. D3 surfaces drift. All three pay the agent cost.

## A3.2 — Question

**Does**: Generate framings with ≥+2σ deviation from baseline (D1: +5σ).
**Tools**: Sonnet/Opus + Cognitive Engines (Voltage Reactor, Frame Collision).
**Model**: Sonnet (Opus at L6).
**Cost**: $0.10 – $0.50.

```python
# a3_2_question.py
from lattice.integrations import dv_engines

def frame_questions(intent: Intent, diamond: Diamond) -> QuestionSet:
    sigma_target = {Diamond.D1_DISCOVERY: 5.0, Diamond.D2_SOLUTION: 2.0, Diamond.D3_EVOLUTION: 0.0}[diamond]

    agent = BudgetedAgent(
        model="claude-sonnet-4-6",
        tools=[phronema_search, dv_engine_score, frame_collision],
        max_iterations=10,
        max_cost_usd=1.5,
    )

    candidates = agent.run(QUESTION_FRAMING_PROMPT, intent=intent)

    # Filter to questions that meet sigma deviation target
    deviated = []
    for q in candidates:
        scores = dv_engines.score_text(q.text)
        if scores["rig_l"] >= sigma_target:
            deviated.append(q)

    if not deviated:
        # Force agent to re-frame more aggressively
        return frame_questions(intent, diamond)  # one retry, then escalate to A4.1
    return QuestionSet(questions=deviated)
```

**Doctrine**: D1 demands +5σ minimum (genuinely contrarian framings). D2 settles at +2σ. D3 baselines (drift detection).

## A3.3 — Research

**Does**: Multi-source agent loop with source-per-claim enforcement + Mem0 long-term memory.
**Tools**: Sonnet + Firecrawl + Jina + Consensus + arXiv + Phronema MCP + Mem0.
**Model**: Sonnet.
**Cost**: $0.30 – $2.00.

```python
# a3_3_research.py
from lattice.integrations import dv_engines

def research(questions: QuestionSet, diamond: Diamond) -> ResearchPack:
    tools = TOOL_RING_BY_DIAMOND[diamond]
    # D1: all 5 MCPs. D2: peer-reviewed only (consensus, arxiv, internal). D3: baseline lookups.

    agent = BudgetedAgent(
        model="claude-sonnet-4-6",
        tools=tools,
        memory=Mem0(namespace=f"a3_research/{diamond.value}"),
        max_iterations=20,
        max_cost_usd=3.0,
        require_source_per_claim=True,  # wrapper enforces — agent CANNOT return unsourced claims
    )

    pack = agent.run(RESEARCH_PROMPT, questions=questions)

    # DV pull on assembled corpus
    pack.dv_packet = dv_engines.score_text(pack.normalized_text())

    # D3 also pulls baseline for comparison
    if diamond == Diamond.D3_EVOLUTION:
        pack.baseline = phronema.fetch_baseline(window_days=90)

    return pack
```

**Doctrine**: `require_source_per_claim=True` is the architectural enforcement. If the agent tries to emit a claim without a source URI, the wrapper rejects it and forces the agent to retry. Source-per-claim is NOT a prompt instruction — it's a Python wall.

## A3.4 — Solution

**Does**: Generate-then-filter-then-judge tournament (5 candidates per diamond).
**Tools**: Sonnet/Opus generators + LLM-as-judge + Cognitive gate filter.
**Model**: Sonnet (Opus at L6+ for D2 only).
**Cost**: $0.20 – $1.50.

```python
# a3_4_solution.py
def solve(research: ResearchPack, diamond: Diamond) -> Solution:
    model = "claude-opus-4-7" if diamond == Diamond.D2_SOLUTION else "claude-sonnet-4-6"

    candidates = generate_n(
        model=model,
        prompt=SOLUTION_PROMPT.format(diamond=diamond.value),
        research=research,
        n=5,
        temperature={Diamond.D1_DISCOVERY: 0.8, Diamond.D2_SOLUTION: 0.3, Diamond.D3_EVOLUTION: 0.1}[diamond],
    )

    # Filter by Cognitive gates
    candidates = [c for c in candidates if cognitive_gates.passes(c, diamond=diamond)]
    if not candidates:
        raise EscalationRequired("no candidate passed Cognitive gates")

    # Tournament: pairwise judge until one survives
    winner = sonnet_judge.tournament(candidates, rubric=SOLUTION_RUBRIC, diamond=diamond)

    # D2 MUST attach mechanism map
    if diamond == Diamond.D2_SOLUTION:
        winner.mechanism = sonnet.generate(MECHANISM_PROMPT, solution=winner, schema=MechanismMap)

    return winner
```

**Doctrine**: 5 candidates → cognitive gate filter → tournament. D2 always uses Opus (highest stakes). D2 attaches mechanism map at this step (not just at Proof) so quality gates can validate it.

## A3.5 — Quality

**Does**: Full gate stack + red-team adversarial attacks + Promptfoo evals.
**Tools**: NeMo Guardrails + Sonnet red-team + Promptfoo + lattice predictions.
**Model**: Sonnet.
**Cost**: $0.10 – $0.50.

```python
# a3_5_quality.py
from lattice.integrations import predictions

def quality(solution: Solution, diamond: Diamond) -> QualityResult:
    # Gate stack — hard-fail on any short-circuit
    gates = run_gate_stack(solution, diamond=diamond)
    if not gates.passed:
        return FAIL(gates)

    # Red-team adversarial attacks
    attack_modes = ATTACK_MODES_BY_DIAMOND[diamond]
    # D1: diversity attacks (echo chamber, premature convergence)
    # D2: ALL attacks (5-20 modes — full sweep)
    # D3: regression attacks (replay last-30d, check Brier drift)
    red_team = sonnet_red_team.attack(solution, attack_modes=attack_modes)
    if red_team.breaks > 0:
        return FAIL(red_team)

    # Predictive challenge layer
    pred_packet = predictions.predictive_swarm(
        seed=solution.body,
        horizon={Diamond.D1: "14d", Diamond.D2: "60d", Diamond.D3: "180d"}[diamond],
        diamond=diamond,
    )

    # D2 and D3 hard requirements
    if diamond == Diamond.D2_SOLUTION and pred_packet["mirofish"]["overall"] < 8.5:
        return FAIL("prediction.mirofish_threshold")
    if diamond == Diamond.D3_EVOLUTION and not pred_packet["milkyway"].get("endorses_patch", False):
        return FAIL("prediction.milkyway_endorsement")

    return PASS(gates=gates, red_team=red_team, predictions=pred_packet)
```

**Doctrine**: Red-team attacks are MANDATORY at A3. D1 gets diversity attacks; D2 gets the full 20-mode sweep; D3 gets regression attacks. Sonnet plays both attacker and defender.

## A3.6 — Proof

**Does**: Falsification charter + mandatory mechanism map + Phronema audit.
**Tools**: Sonnet + Pydantic + Neo4j + Phronema audit log.
**Model**: Sonnet (charter + mechanism map generation).
**Cost**: $0.05 – $0.20.

```python
# a3_6_proof.py
def proof(solution: Solution, research: ResearchPack, diamond: Diamond) -> ProofPacket:
    charter = sonnet.generate(
        CHARTER_PROMPT,
        solution=solution,
        diamond=diamond,
        schema=FalsificationCharter,
    )

    # Mechanism map is MANDATORY at A3 — A1 didn't have this, A2 had it lightweight
    mechanism = sonnet.generate(
        MECHANISM_PROMPT,
        solution=solution,
        schema=MechanismMap,
    )

    # Validate that mechanism has at least 3 nodes + 2 edges (no trivial mechanisms)
    if len(mechanism.nodes) < 3 or len(mechanism.edges) < 2:
        raise ProofValidationError("mechanism map too sparse for A3")

    return ProofPacket(
        charter=charter,
        mechanism=mechanism,
        evidence=research.normalized(),
        gates=quality.gates,
        red_team=quality.red_team,
        predictions=quality.predictions,
        graph_node=neo4j.write_node(solution, mechanism=mechanism),
        diamond=diamond,
        signer="a3.agent_bounded.v1",
    )
```

**Doctrine**: Mechanism map is non-negotiable. Sparse mechanisms (< 3 nodes / 2 edges) fail at validation — agent is forced to elaborate or escalate.

## A3.7 — Integrate

**Does**: MANDATORY human approval. No auto-approve at this mode or above.
**Tools**: AionUI + Postgres append-only audit log.
**Model**: None (the decision is human).
**Cost**: ~$0.

```python
# a3_7_integrate.py
async def integrate(packet: ProofPacket) -> Decision:
    # D1 Discovery: audit-only, never dispatched even after approval
    if packet.diamond == Diamond.D1_DISCOVERY:
        audit.record(packet, decision=None, kind="discovery_only")
        return BLOCKED("D1 discovery — no dispatch by doctrine")

    decision = await aionui.request_approval(
        packet,
        require_signoff=True,
        timeout="72h",
        show_mechanism_map=True,
        show_falsification_charter=True,
        show_red_team_results=True,
        show_predictions=True,
    )
    audit.record(packet, decision)

    if not decision.approved:
        return BLOCKED(decision.reason)

    result = await execute(packet, signed_by=decision.signer)
    audit.record(packet, action=result.action, result=result)
    return result
```

**Doctrine**: AionUI shows everything: mechanism, charter, red-team, predictions. 72-hour timeout (not 24h like A2). D1 is dispatch-blocked by doctrine — discovery cells generate proof but never act.

## End-to-End Flow

```python
async def run_a3(payload: InboundPayload, diamond: Diamond) -> Decision:
    intent = extract_intent(payload, diamond=diamond)
    questions = frame_questions(intent, diamond=diamond)
    research = await research_step(questions, diamond=diamond)
    solution = solve(research, diamond=diamond)
    quality_result = quality(solution, diamond=diamond)
    if not quality_result.passed:
        return await escalate_to_a4(payload, diamond=diamond, reason=quality_result.reason)
    packet = proof(solution, research, diamond=diamond)
    return await integrate(packet)
```

**One sentence**: `extract_intent (agent loop, budgeted) → frame_questions (+Nσ deviation) → research (source-per-claim agent + Mem0) → solve (5-candidate tournament + mechanism) → quality (gate stack + red-team + predictions) → proof (charter + mechanism map mandatory) → integrate (AionUI 72h signoff mandatory)`.
